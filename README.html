ğŸš€ Installation & Run Steps

1Clone the Repository

git clone https://github.com/Priyanshu23146/Smart-Solar-Energy-Dashboard.git cd Smart-Solar-Energy-Dashboard

2Install All Dependencies

npm run install:all

This installs dependencies for both frontend and backend.

3Add Weather API Key (Optional)

Create a .env file in the root directory:

VITE_WEATHER_API_KEY=your_api_key_here

Note: The app works with mock data if no API key is provided.

4Run the Project

npm run dev

This starts both the frontend (Vite) and backend (JSON Server) concurrently.

5Open in Browser

Frontend: http://localhost:5173 Backend API: http://localhost:3000

6Create an Account

Navigate to the signup page, create your account with a username, password, and city for weather data.
